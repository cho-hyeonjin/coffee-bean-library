React Context API를 이용하여 상태관리를 구현하였고, 서버 상태관리는 Tanstack Query를 이용했습니다.

DB, 배포, 로그인 인증은 Firebase를, CSS는 Tailwindcss를 이용하여 구현한 개인 프로젝트입니다.
